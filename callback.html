OAuth callback received

If you see a URL like ?code=...&state=netmax, copy the code value and paste it into your token exchange command.

Example (PowerShell/cmd):

curl -X POST "https://oauth.platform.intuit.com/oauth2/v1/tokens/bearer" ^
  -u "CLIENT_ID:CLIENT_SECRET" ^
  -H "Accept: application/json" ^
  -H "Content-Type: application/x-www-form-urlencoded" ^
  -d "grant_type=authorization_code&code=PASTE_CODE_HERE&redirect_uri=https://netmaxpc.github.io/netmaxpass/callback.html"

If you prefer a custom domain, use the same page at https://netmaxpass.com/callback.html and update the redirect_uri accordingly.
